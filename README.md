
# 📊 PhonePe Payment Insights Dashboard

## 📌 Project Overview
This project presents an **interactive Power BI dashboard** built to analyze PhonePe’s digital payment ecosystem. The dashboard provides insights into transactions, user demographics, service performance, and usage patterns, enabling data-driven decision-making.

---

## 🛠 Tools & Skills Used
- **Power BI** – Data modeling, DAX, interactive visuals  
- **Data Cleaning & Transformation**  
- **Dashboard Design & Storytelling with Data**  

---

## 🔑 Key Features
- **KPIs Overview** – Transactions, Users, Success Rate  
- **Trends Analysis** – Monthly transaction growth patterns  
- **User Demographics** – Age segment contribution (Millennials, Gen X, Gen Z, Boomers)  
- **Service Analysis** – Loans, Insurance, Money Transfer, Recharge  
- **Usage Patterns** – Weekday vs Weekend engagement  
- **Top Users** – High-value contributors  

---

## 📈 Insights
- Steady growth in transactions and user adoption  
- Millennials & Gen X dominate usage  
- Loans drive majority of service value  
- Higher engagement during weekdays  

---

## 🎯 Conclusion
The dashboard demonstrates how **Power BI can transform raw data into actionable insights** for digital payment platforms. It highlights trends, user behavior, and service performance, supporting strategic business decisions.
